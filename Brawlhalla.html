¿Que es Brawlhalla?

Este es un juego que se basa en combates en plataforma 2D, este juego se puede jugar con un maximo de 4 personas.

Partidas casuales:

Este tiene distintos modos de juego como: Todos contra todos(4 players), Strikeout 1vs1 donde se tienen que elegir 3 leyendas para jugar, Amistosos 2vs2 y Experimental 1vs1, en este solo se usa una leyenda.

Partidas competitivas:

Tambien tiene modo competitivo, con distintos tipos de variaciones como: competitivo 1vs1, competitivo 2vs2 (este se tiene que jugar si o si con un amigo) y Futbol peleón 2vs2, este se basa en meter la pelota en el arco del enemigo mientras los otros intentan pegarte para quitarte.

Partidas sin conexion:

En las partidas sin conexion puedes jugar: El modo local (este es una sala personalizada), Entrenamiento (sirve para practicar combos), Torneo (torneo de partidas todos contra todos con bots) y Tutoriales (simplemente tutoriales)
